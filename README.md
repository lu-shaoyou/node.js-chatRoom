# -基于node.js的即时聊天室-
## 实现私聊跟群聊
## 如何运行
### 下载项目
`git clone git@github.com:lucky-oss/-node.js-.git`
### 进入项目安装依赖
`npm install`
### 运行项目
`node app.js`
### 运行成功后访问localhost:2020
